基于node爬虫的阴阳师贴吧讨论热度词云

# 快速开始
```
npm start
```
打开`index.html`

# 技术栈
- 词云canvas生成：[wordcloud2](https://github.com/timdream/wordcloud2.js/)
- 分词：[nodejieba](https://github.com/yanyiwu/nodejieba)
- node并发控制：[async.js](https://github.com/caolan/async)
